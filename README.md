# 📩 Spam Detection using NLP

## 📌 Overview
This project detects whether a message is Spam or Not Spam using Machine Learning.

## ⚙️ Technologies Used
- Python
- Scikit-learn
- NLP (TF-IDF)
- Streamlit

## 🚀 How to Run
py -m streamlit run app.py

## 🧠 Model Used
- TF-IDF Vectorization
- Multinomial Naive Bayes

## 📊 Accuracy
~96%

## 🧪 Example
Input: "Win ₹5000 now!!!"  
Output: SPAM
